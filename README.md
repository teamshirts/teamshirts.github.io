teamshirts.github.io
=============

Website for teamshirts.de

Developing
===

Mac / Linux
---

* clone the repository
* `cd teamshirts.de`
* install jekyll with `gem install jekyll`
* run `jekyll serve --watch`

Windows
---

* install VirtualBox
* install Vagrant
* clone the repository
* `cd teamshirts.de`
* start the vm with `vagrant up`
* open http://192.168.33.10:4000
